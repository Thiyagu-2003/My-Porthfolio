## 🧭 About the Project

Welcome to my **Cloud & DevOps Portfolio Website** — a responsive and modern web app built with **React**, **TypeScript**, and **Vite**.  
It showcases my journey, certifications, cloud skills, and real-world projects in **AWS**, **Linux**, **Docker**, and **CI/CD** automation.

> 🚀 Live Demo: [https://teofila-unscaling-subspirally.ngrok-free.dev](https://teofila-unscaling-subspirally.ngrok-free.dev)

## ✨ Features

- ⚡ Built with **Vite + React + TypeScript** for performance and scalability  
- 🎨 **Tailwind CSS** styling with minimal, responsive design

- 📱 **Mobile-friendly** responsive layout

## 🏗️ Installation & Setup

Clone the repository:
```bash
git clone <your-repo-url>
cd thiyagu-cloud-portfolio
````

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
